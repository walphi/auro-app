## 5. SEO Formatting Rules

| Element | Rule |
|---|---|
| **Meta title** | `{Primary Keyword} | AURO` — max 60 characters |
| **Meta description** | Action verb + primary keyword + benefit + CTA — max 160 characters |
| **Canonical URL** | Trailing slash, no `www`, lowercase |
| **OG image** | `https://auroapp.com/auro-og.jpg` — 1200×630 px |
| **robots** | `index, follow` (default); `noindex` for `/dashboard`, `/client-login`, `/brief/*` |
| **JSON-LD types** | `Organization` (all pages), `BlogPosting` (articles), `BreadcrumbList` (subpages), `FAQPage` (faq) |
| **H1** | One per page, matches primary keyword |
| **URL slug** | Lowercase kebab-case, primary keyword first |
